run_block module¶

run_block.checkBlockEnd(len_num, end)¶

检查代码块结束的合法性。开始指针不移动

参数:

  • len_num (integer) – 现在读取到哪一行，用于进行报错处理

  • end (integer) – 语句块终止的位置的下一位。语句块结束后，下一个语句的起始位置。

返回:

  • None 代表检测到错误

  • 1 代表执行语句成功

run_block.runBlock(len_num, end)¶

运行if, while, switch代码块，返回后的指针指向下一个待执行的代码块

参数:

  • Args –

  • len_num (integer) – 现在读取到哪一行，用于进行报错处理

  • end (integer) – 语句块终止的位置的下一位。语句块结束后，下一个语句的起始位置。

返回:

  • None 代表检测到错误

  • 其他 代表块执行成功，由于这里需要执行if, while, switch以及普通语句，所以返回值类型比较多
    • 普通语句

      • 若为赋值语句，返回赋值语句返回的值

      • 若不为赋值语句则返回1

    • 其他语句，则是使用了递归调用进行语句块的执行。
