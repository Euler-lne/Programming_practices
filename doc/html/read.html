class read.Read(path, encodeing='utf-8')¶

读取文件类，词法分析边读边生成Token，以一个语法块为一个单位。

创建Token

参数:

char (char) – 读取到的字符

返回:

  • None 代表检测到错误

  • 1 代表执行语句成功

检查相关的字符，然后加入到Token中

参数:

char (char) – 读取到的字符

返回:

  • None 代表检测到错误

  • 1 代表执行语句成功

checkID(is_swap=True)¶

检查是否为ID，由于ID没有特定的标识， 所以不为buildToken中的所有汉字都有可能为ID。 在两个能匹配项之间的汉字记为ID，能匹配指的是处于const.token的 当buildToken检测匹配时才调用这个函数进行添加ID

参数:

is_swap (bool, optional) – 用于判断Token是否要交换位置. Defaults to True.

checkString(char)¶

完成字符串检查，通过控制栈来实现

参数:

char (char) – 读取到的字符

divideToken()¶

将现有的Token划分未块，起到一个检测块边界的问题 由于id 采用特殊的匹配方式，如果没有匹配到id那么Token不会增长，导致if会多次入栈。 所以，divideToken函数之后在Token改变了才调用

返回:

  • None 代表检测到错误

  • 1 代表执行语句成功

readBlock()¶

读取文件，边读边生成Token，保存到const.token中。 每次读取到逗号，如果有分支或者循环读取到相应的分支或者循环的终！

返回:

  • None 代表检测到错误

  • 其他 代表执行语句成功，对应相应的代码状态

saveToken(path='./log/token', encoding='utf-8', col=5)¶

将Token 保存到指定路径，指定编码，指定列数

参数:

  • path (str, optional) – 保存路径. Defaults to “./log/token”.

  • encoding (str, optional) – 中文编码. Defaults to “utf-8”.

  • col (int, optional) – 行数. Defaults to 5.
