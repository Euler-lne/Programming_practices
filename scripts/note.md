### 语法规则：
* 定义变量
```
有数曰：苹果为3，香蕉为4。
有分曰：气温为13.5。
有言曰：询问为“1234”。
有爻曰：真为阳，假为阴。
```
* 条件
```
若1加1同2者，则苹果加1也，非者苹果减1也。
```
* 循环
```
凡苹果非同1者，则苹果减1也。
```
* 打印
```
曰：“此乃妙哉”。
```
* 分支
```
苹果者：
若为1，则……，终；
若为2，则……，终。
```
